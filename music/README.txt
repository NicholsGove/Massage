MUSIC SETUP
===========

Drop your 7 songs in THIS folder, named exactly:

  song1.mp3
  song2.mp3
  song3.mp3
  song4.mp3
  song5.mp3
  song6.mp3
  song7.mp3

That's it. When Renei taps the welcome screen to enter, the songs
play automatically in order and loop forever (song7 -> back to song1).

Notes:
- Use .mp3 files (most compatible across browsers).
- If a song is missing, the player just skips to the next one,
  so the site still works while you're still adding tracks.
- A small ♪ button (top-right) lets her mute/unmute if she wants.
